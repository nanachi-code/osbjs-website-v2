# Easing functions
Easing functions specify the rate of change of a parameter over time.

All of the easing functions take 1 parameter which is the time progress between 0 (start) and 1 (end).

Available easing functions:

- `easeInSine`
- `easeOutSine`
- `easeInOutSine`
- `easeInCubic`
- `easeOutCubic`
- `easeInOutCubic`
- `easeInQuint`
- `easeOutQuint`
- `easeInOutQuint`
- `easeInCirc`
- `easeInElastic`
- `easeOutElastic`
- `easeInOutElastic`
- `easeInQuad`
- `easeOutQuad`
- `easeInOutQuad`
- `easeInQuart`
- `easeOutQuart`
- `easeInOutQuart`
- `easeInExpo`
- `easeOutExpo`
- `easeInOutExpo`
- `easeInBack`
- `easeOutBack`
- `easeInOutBack`
- `easeInBounce`
- `easeOutBounce`
- `easeInOutBounce`
